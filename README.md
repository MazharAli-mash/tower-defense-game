# Tower Defense Game

## Description
A strategic console-based tower defense game where players build defensive towers to protect against waves of enemies.

## Features
- Tower placement and upgrades
- Enemy waves system
- Map generation and import/export
- Loot and inventory system
- In-game store with virtual currency
- Achievement system
- Experience and progression
- Player leaderboard

## Getting Started
### Requirements
- C++17 or higher
- Visual Studio Code
- C++ compiler (MinGW/MSVC on Windows, GCC on Linux, Clang on Mac)

### Build & Run
```bash
g++ src/*.cpp -o tower-defense-game
./tower-defense-game
